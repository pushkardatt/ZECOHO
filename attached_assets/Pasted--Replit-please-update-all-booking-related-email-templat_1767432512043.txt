SCOPE
=================================

- Property Owner

This includes:
- Booking request created

=================================
MANDATORY DETAILS TO INCLUDE
=================================

Every booking email MUST include the following sections clearly:

---------------------------------
PROPERTY DETAILS
---------------------------------
- Property Name
- Full Property Address (street, locality, city, state)
- Property Contact Number
- Property Location link (Google Maps URL using lat/long)

---------------------------------
ROOM DETAILS (CRITICAL – CURRENTLY MISSING)
---------------------------------
- Room Type (e.g. Deluxe, Premium, Family)
- Room Description (if available)
- Max Occupancy
- Number of Rooms Booked
- Meal Plan / Room Option (Room only / Breakfast / etc.)

---------------------------------
BOOKING DETAILS
---------------------------------
- Booking ID
- Booking Date (created_at)
- Check-in Date
- Check-out Date
- Number of Nights
- Guest Count (Adults / Children)
- Booking Status (Pending / Confirmed / Checked-in / Completed)

---------------------------------
PRICE DETAILS
---------------------------------
- Price per night
- Total Amount
- Token / Pay@Hotel amount (if applicable)
- Payment Mode (Pay at Hotel / Advance Paid)
